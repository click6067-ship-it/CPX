# PPI(환자의사관계) 평가 루브릭

> PPI는 이분법으로 환원 불가(architecture §3-④, Codex R1-10). **관찰가능한 미시행동 + 등급**으로.
> ④채점의 PPI domain은 이 루브릭을 사용(병력청취·신체진찰은 이분법 유지). debrief.py(유도성·전문용어)와 연결.

## 미시행동 (관찰 가능 — AI가 발화 근거와 함께 채점)
| 영역 | 행동 지표 | 등급 |
|---|---|---|
| 라포·존중 | 인사·자기소개·환자 호명·예의 | 0 없음 / 1 일부 / 2 충분 |
| 공감 | 환자 감정 인정 표현("힘드셨겠어요") | 0 / 1 형식적 / 2 맥락적 |
| 경청 | 말 끊지 않음, 되짚기(reflection) | 0 / 1 / 2 |
| 면담 구조화 | 면담주제 협상, 개방형→폐쇄형 순서 | 0 / 1 / 2 |
| 이해 확인 | 환자 이해도 확인·요약·질문 기회 | 0 / 1 / 2 |
| 쉬운 설명 | 전문용어 회피(일상어) | 0 / 1 / 2 |

## 채점 방식
- AI는 **각 미시행동을 관찰가능 증거(발화 인용)와 함께 0/1/2** 로 제시. "진정성" 단정 회피.
- 진정성·정서 같은 **주관 영역 = AI 보조 + 인간(교수) 최종** (국시 4단계 평가 대체 ❌).
- debrief.py의 `is_leading`·`uses_jargon` 플래그가 '면담 구조화'·'쉬운 설명' 근거로 직결.

## 검증 (H4-real)
- PPI 일치도는 grader보다 어려움(주관) → **PPI는 별도 kappa + 인간-인간 floor 먼저**. 과대주장 금지.
